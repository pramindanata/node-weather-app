# Node Weather App
Simple weather app based on "The Complete Node.js Developer Course (2nd Edition)".

## Instalation
1. Install node modules.
    ```bash
    npm install
    ```

2. Run the app.
    ```bash
    node app.js -a "Any address"
    ```

    Or run this if you prefer promise version.
    ```bash
    node app-promise.js -a "Any address"
    ```
